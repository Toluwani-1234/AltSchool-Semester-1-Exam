This Weather App was built using HTML, CSS, and JavaScript. It allows users to search for a city and view current weather conditions along with a 5-day forecast. The interface is designed to be simple and user-friendly, displaying key weather details such as temperature, humidity, wind speed, and UV index. The app structure is built with semantic HTML, styled using CSS for layout and responsiveness, and powered by JavaScript to handle dynamic data rendering and user interactions. Weather data is intended to be fetched from a weather API, enabling real-time updates based on user input.
